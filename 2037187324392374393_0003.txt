# BCM4331 Wi-Fi Fixer for Fedora 43

One-click, deterministic fix for the **Broadcom BCM4331** (`14e4:4331`) wireless chipset on Fedora 43 when the open-source `b43` driver fails.

## Why this exists
- The open `b43` + `b43-openfwwf` firmware is incomplete for the BCM4331 on Fedora 43.
- `akmod-wl` (proprietary Broadcom driver) works perfectly but requires a kernel that has matching `kernel-devel`.
- Your system was stuck on kernel 6.17.7 (no devel package available) while `akmod-wl` was already built for 6.19.8.
- Wired Ethernet (`tg3`) is in-tree and survives the kernel switch.

## What the script does (safely)
1. Installs the newer kernel + matching devel packages (old kernel remains).
2. Rebuilds `akmod-wl` for the new kernel.
3. Removes conflicting open drivers.
4. Reboots (you stay in control — GRUB lets you pick the old kernel if needed).
5. Loads the `wl` driver and verifies both wired **and** wireless interfaces.

**Wired networking is guaranteed to work after reboot** — `tg3` is compiled into every Fedora kernel.

#!/usr/bin/env bash
# bcm4331_diagnostics.sh - Collect everything needed to debug BCM4331

set -euo pipefail

LOGDIR="$HOME/bcm4331_diag_logs"
mkdir -p "$LOGDIR"

echo "=== BCM4331 Diagnostics ===" | tee "$LOGDIR/00_summary.log"

echo "STEP 1: PCI devices" | tee -a "$LOGDIR/00_summary.log"
lspci -nnk | grep -iA4 "Network\|Wireless\|Broadcom" > "$LOGDIR/step1_pci.log"
cat "$LOGDIR/step1_pci.log"

echo "STEP 2: Loaded modules" > "$LOGDIR/step2_lsmod.log"
lsmod > "$LOGDIR/step2_lsmod.log"
cat "$LOGDIR/step2_lsmod.log" | head -20

echo "STEP 3: Driver info" > "$LOGDIR/step3_drivers.log"
for mod in wl b43 bcma ssb tg3; do
    echo "=== $mod ===" >> "$LOGDIR/step3_drivers.log"
    modinfo "$mod" 2>/dev/null || echo "Module $mod not found" >> "$LOGDIR/step3_drivers.log"
done

echo "STEP 4: Network interfaces" > "$LOGDIR/step4_interfaces.log"
ip link > "$LOGDIR/step4_interfaces.log"
nmcli device status >> "$LOGDIR/step4_interfaces.log"

echo "STEP 5: Firmware" > "$LOGDIR/step5_firmware.log"
ls -lR /lib/firmware/b43* 2>/dev/null || echo "No b43 firmware found" > "$LOGDIR/step5_firmware.log"

echo "STEP 6: dmesg wireless" > "$LOGDIR/step6_dmesg.log"
sudo dmesg | grep -E 'b43|wl|bcma|broadcom|4331' > "$LOGDIR/step6_dmesg.log" || true

echo "All logs written to $LOGDIR"
echo "Zip and attach $LOGDIR if you open an issue."
